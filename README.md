# PortSwigger Web Security Academy - XSS Labs

## Overview

This repository contains a detailed writeup of 9 Apprentice-level Cross-Site Scripting (XSS) labs completed on PortSwigger Web Security Academy.

The objective of this project was to gain hands-on experience in identifying, exploiting, and understanding different types of XSS vulnerabilities in a controlled lab environment.

---

## Topics Covered

- Reflected XSS
- Stored XSS
- DOM-Based XSS
- HTML Context Injection
- Attribute Context Injection
- JavaScript Context Injection
- jQuery-based DOM XSS
- XSS Payload Analysis
- Secure Coding Concepts
